# Expense-Tracker-Python
A Python-based Expense Tracker that records expenses, calculates total spending, and generates a summary report using loops, lists, and basic mathematical operations.
# Expense Tracker Using Python
A simple Expense Tracker developed using Python that allows users to record multiple expenses and calculate total spending.

## Features
- Add multiple expenses
- Calculate total amount spent
- Display expense summary
- Count total number of expenses

## Technologies Used
- Python
- Anaconda
- Jupyter Notebook

## Concepts Applied
- Variables
- Lists
- While Loops
- Conditional Statements
- User Input Handling
- Mathematical Operations
